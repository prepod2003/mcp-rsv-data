# MCP:RSV Data

Бесплатный MCP-сервер для работы с **данными рабочей базы 1С:Предприятие** (запросы,
чтение объектов, остатки, отчёты) для AI-ассистентов — Claude, Cursor и др. Реализован как
расширение конфигурации 1С.

> 🚧 **Скоро.** Репозиторий создан заранее; код и инструкции появятся к релизу.

## Разработка в 1С:EDT

Нужен AI-инструмент для разработки конфигураций и расширений 1С (код, метаданные, формы,
СКД, отладка)? Это профессиональный продукт линейки RSV — **MCP:RSV Server**:
[prepod2003.github.io/edt-ai-1c](https://prepod2003.github.io/edt-ai-1c)
